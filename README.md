# Texture List Generator

Generates a list of textures (png, jpg, gif) under the current folder (recursively) in html.

# Download

https://github.com/t-kuni/texture-list-generator/releases

# Usage

```bash
texture-list-generator
```

# For Developer

## Requirements

* go 1.18+

## Build

```bash
make build
```

## Run

```bash
go run ./main.go
```

## Run test

```bash
make test
```
